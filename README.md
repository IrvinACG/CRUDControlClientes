# CRUDControlClientes
Es un CRUD de Persona realizado en Java para el backend utilizando el framework SpringBoot, usando como Gestor de Base de Datos MySQL.

## Tecnologias usadas

->Spring MVC

->Thymeleaf

->Spring Data

->Spring Security

->Bootstrap
